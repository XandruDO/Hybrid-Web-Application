Instructions

This is a simple next generation Calculator Application with a minimalist design for on the go quick and simple calculations.

One of its key features is that you can scroll up and down the values you have entered to both double check them as well as be able to change them, in case you might have entered the wrong number in the process.

As well as the Calculator itself, the second page (as mentioned, top right corner button to go to the page), the second page offers a way to contact its creator, Alexandru D. Opre, for any complaints, suggestions or anything else you can think of related to the app.

Go ahead! Push that button on the top right corner of the app and lets do some calculations!

Thank you for using this NextGEN Calculator App!
